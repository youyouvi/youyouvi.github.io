Go程序的一般结构

1、Go程序是通过package组织的;

2、只有package为main的包才能有main(),即一个程序有且只有一个package main和一个main()

/* 多行注释 */
// 当前程序的包名,必须放在首行
package main

// 导入其它包
import (
	"fmt"
	// 为导入的包设置别名: 别名 "包名";当别名设为"."时,可以直接调用其函数(不建议,容易混淆)
	sys "os"
)

// 定义常量
const PI  = 3.14

// 全局变量定义,可用于整个包内
var name = "go"

// 一般类型声明
type newType int

// 结构声明
type gopher struct {}

// 接口声明
type golang interface {}

// 由main作为程序的启动点
func main() {
	// 导入包后必须使用,否则编译出错,调用方法: packageName.funcName
	sys.Hostname()
	fmt.Println("Hello " + name + " !")
}

3、导入多个包时可以简写,声明多个常量、全局变量或一般类型（非接口、非结构）时也可以简写

// 常量定义
const (
	PI = 3.14
	const1 = "1"
	const2 = 2
)

// 全局变量的声明与赋值,var组只能用于声明全局变量,函数内部是不可以使用的
var (
	name = "go"
	name1 = "1"
	name2 = 2
)

// 一般类型声明
type (
	newType int
	type2 float32
)

Go可见性规则

Go中使用大小写决定常量、变量、类型、结构、接口或函数是否可以被外部包调用；

函数名首字母小写即为private，首字母大写即为public。

数据类型的零值

零值不等于空值，而是变量被声明为某种数据类型后的默认值：一般值类型的默认值为0，bool为false，string为空字符串。

package main

import (
	"fmt"
	"math"
)

// 为数据类型设置别名
type (
	文本 string

)
func main() {
	var  a int
	// 获取数据类型默认值
	fmt.Println(a)
	// 获取数据类型最大值
	fmt.Println(math.MaxInt8)
	var b 文本
	b = "中文类型名"
	fmt.Println(b)
}

单个变量的声明与赋值

变量的声明格式: var value_name value_type

变量的赋值格式: value_name = 表达式

声明的同时赋值: var value_name [value_type] = 表达式 (多用于全局变量)

变量的简短声明: value_name := value （多用于局部变量）

package main

import "fmt"

func main() {
	var v1 int
	v1 = 1
	// 可省略数据类型,由系统自动根据赋值进行判断
	var v2 = 1
	// 赋值简短方式,省略var和类型 value_name := value
	v3 := 1
	fmt.Println(v1)
	fmt.Println(v2)
	fmt.Println(v3)
}

多个变量的声明与赋值

1、全局变量的声明可以使用var()方式

2、简写全局变量的声明不可以省略var,但可使用并行方式

3、所有变量都可以使用类型推断

4、局部变量不能使用var()方式简写，只能使用并行方式

package main

import "fmt"

var (
	// 常规方式
	aaa = "hello"
	// 并行方式
	sss, bbb = 1, 2
	// ccc := 3 不可省略var,因此该方式无法使用
)


func main() {
	// 多个变量声明
	var a, b, c, d int
	// 多个变量赋值
	a, b, c, d = 1, 2, 3, 4
	// 多个变量声明的同时赋值
	var e, f, g int = 5, 6, 7
	// 省略变量类型,由系统推断
	var i, j, k  = 9, 10, 11
	l, m, n := 13, 14, 15
	fmt.Println(a,b,c,d,e,f,g,i,j,k,l,m,n)
}

变量的类型转换

1、go中不存在隐式转换,所有类型转换必须显式声明

2、转换只能发生在两种相互兼容的类型之间

3、类型转换的格式: valueA = valueA_type(valueB)

package main

import "fmt"

func main() {
	// 相互兼容的两种类型进行转换
	var a float32 = 1.1
	b := int(a)

	// 不兼容的类型无法转换
	var c bool = true
	d := int(c)
	fmt.Println(b)
	fmt.Println(d)
}
